# SocialInfluence

Author : Blackliquid

######################################################################################################

Experimenting with the greedy algorithm from "Maximizing the Spread of Influence through a Social
Network" by Kempe/Kleinberg/Tardos

######################################################################################################

Trying out the greedy algorithm from the mentionned paper on a small social network  (anonymized facebook), comparing the performance to the optimal solution.

Programmed in Julia. Written in iPython notebooks/IJulia.

To launch a script in Julia use > include("filepath")

#######################################################################################################

To install iJulia, open Julia in the command line and type 

>Pkg.add("IJulia")

To open the iPython notebook, type

>using IJulia
>notebook()
